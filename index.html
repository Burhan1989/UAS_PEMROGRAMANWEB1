SOAL UAS PEMROGRAMAN WEB 1 :

1. Jelaskan cara kerja Website serta jelaskan hubungan antara Web Server, Domain, dan Hosting ! Jawaban dibuat di dalam halaman/layout soal nomor 2 ! (20 point)

Jawab :

a. Cara Kerja Website : Sebuah website bekerja dengan cara menerima dan merespons permintaan dari pengguna melalui browser (seperti Chrome, Firefox, Safari). Setiap kali Anda mengetikkan alamat sebuah website atau mengklik link, browser Anda akan mengirim permintaan ke server tempat website tersebut disimpan. Server kemudian akan merespons dengan mengirimkan file-file yang dibutuhkan oleh browser Anda untuk menampilkan website tersebut. File-file ini biasanya berupa kode HTML, CSS, JavaScript, serta media seperti gambar dan video.

b. Web Server : Web server adalah tempat dimana file-file website disimpan. Server ini memiliki peran penting dalam proses komunikasi antara website dan pengguna. Ketika pengguna mengakses website, web server akan mengirimkan file-file yang dibutuhkan oleh browser pengguna untuk menampilkan halaman website tersebut.

c. Domain : Domain adalah alamat unik yang digunakan pengguna untuk mengakses website. dan bisa membeli domain dari penyedia layanan domain. Setiap domain akan dikaitkan dengan alamat IP dari server tempat website Anda disimpan. Alamat IP ini biasanya berupa serangkaian angka yang sulit diingat, sehingga domain menjadi cara yang lebih mudah untuk mengakses website.

d. Hosting : Hosting adalah layanan yang menyediakan ruang di server untuk menyimpan file-file website Anda. Ketika Anda membeli layanan hosting, Anda sebenarnya sedang menyewa ruang di server milik perusahaan hosting. Ada berbagai jenis layanan hosting, seperti shared hosting, VPS hosting, dan dedicated hosting, masing-masing dengan kelebihan dan kekurangannya sendiri.
